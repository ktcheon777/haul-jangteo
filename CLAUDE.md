# 하늘장터 개발 가이드

이 프로젝트는 **개발 공부용** 중고거래 웹사이트입니다.  
당근마켓과 비슷한 서비스를 직접 만들어보며 웹 개발을 배우는 것이 목표입니다.

---

## 기술 스택 (사용 중인 도구들)

| 도구 | 역할 | 쉬운 설명 |
|------|------|-----------|
| **Next.js** | 프론트엔드 프레임워크 | 화면(웹페이지)을 만드는 도구 |
| **Supabase** | 백엔드 + 데이터베이스 | 회원정보·상품 데이터를 저장하는 서버 |
| **Vercel** | 배포 플랫폼 | 만든 웹사이트를 인터넷에 올리는 서비스 |
| **Tailwind CSS** | 디자인 도구 | 버튼, 색상, 레이아웃을 빠르게 꾸미는 도구 |
| **TypeScript** | 프로그래밍 언어 | JavaScript에 오타 방지 기능을 추가한 버전 |

## Claude 행동 규칙

### 커뮤니케이션
- 작업을 시작하기 전에 **무엇을 할 것인지 먼저 한 문장으로 설명**한다
- 중간중간 **지금 무엇을 하고 있는지** 짧게 알려준다
- 개발 용어가 나오면 **괄호 안에 쉬운 말로 설명**을 함께 적는다
  (예: "미들웨어(페이지 이동 전에 실행되는 코드)를 수정할게요")
- 작업이 끝나면 **무엇이 바뀌었는지 한두 문장으로 요약**한다

### 작업 방식
- 사용자가 **지시한 것만** 수행한다 — 요청하지 않은 기능 추가나 리팩토링(코드 정리)은 하지 않는다
- 한 번에 모든 것을 하지 않고 **단계적으로** 진행한다
- 코드를 수정하면 **바로 미리보기(preview)**를 실행해서 결과를 화면으로 확인한다

### 자동으로 처리하는 것 (사용자가 신경 쓰지 않아도 되는 것)
- 코드 파일 작성 및 수정
- Supabase 데이터베이스 테이블 생성 및 SQL 실행
- npm 패키지(라이브러리) 설치
- Git 커밋(저장) 및 GitHub 푸시(업로드)
- Vercel 배포(인터넷에 올리기)
- 개발 서버 실행 및 브라우저 미리보기

### 사람이 직접 해야 하는 것 (Claude가 할 수 없는 것)
- **Supabase 대시보드** 설정 변경 (URL 설정, 이메일 템플릿 등)
  → 이 경우 Claude가 정확한 주소와 입력값을 안내한다
- **결제·계정 관련** 설정 (Vercel, Supabase 요금제 등)
- **비밀번호·보안 키** 직접 입력

### 배포 후 확인
- 코드 수정이 끝나면 **항상** 다음 순서로 진행한다:
  1. 로컬(내 컴퓨터)에서 미리보기로 정상 작동 확인
  2. GitHub에 커밋·푸시
  3. Vercel에 배포
  4. 배포된 주소 안내

## 파일 구조 설명

```
market/
├── app/                            ← 웹페이지 파일들이 있는 폴더
│   ├── page.tsx                    ← 홈 화면 (메인 페이지)
│   ├── layout.tsx                  ← 모든 페이지에 공통으로 적용되는 틀
│   ├── globals.css                 ← 전체 디자인 기본 설정
│   ├── login/page.tsx              ← 로그인 페이지
│   ├── signup/page.tsx             ← 회원가입 페이지
│   ├── auth/
│   │   ├── actions.ts              ← 로그인·회원가입·탈퇴 처리 코드
│   │   └── callback/route.ts       ← 이메일 인증 처리
│   └── components/
│       └── DeleteAccountButton.tsx ← 회원탈퇴 버튼 컴포넌트(부품)
├── lib/supabase/
│   ├── client.ts                   ← 브라우저에서 Supabase 연결
│   └── server.ts                   ← 서버에서 Supabase 연결
├── middleware.ts                    ← 페이지 이동 전에 로그인 확인
├── .env.local                       ← Supabase 연결 키 (비밀 파일, Git에 올리지 않음)
└── CLAUDE.md                        ← 이 파일
```
